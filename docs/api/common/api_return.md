### 返回格式

```json
{
    "code": "1",
    "message": "操作成功",
    "data": {}
}
```

### 返回状态（code）

| 返回状态 |      描述      |
|---------|:-------------:|
| 1       | 成功          |
| 10      | 没有登录       |
| 20      | 未完善资料     |
| -1      | 失败          |

### 返回消息（message）

* code为-1时会有错误描述信息

### 返回数据（data）

* 请求接口所列返回字段均为data属性名
* 无字段表示只返回code和message
